Follow the below steps to run Depth anything in your rasberry Pi:

1. After installing the Gaashub app, check where the GaasHub App image is present.
2. CD to that folder in a terminal.
3. Now, Open GaasHUb app using the command: ./GaasHub.AppImage --no-sandbox
4. App will open and you need to login
5. Click the power button in the app to connect to GPU
6. Now clone this repository
7. In this repository,asset folder have the input. We have provided one image, you can use yours too. Just put the image in the asset folder.
8. Now open new terminal and CD to the folder where you cloned this repository.
9. To run the code, type gaashub . (gaashub space dot), and press enter.
10. It will take few seconds and the depth output image will be generated in a new folder in the same path.
11. Congratulations!!, you have run depth anything algorithm in your rasberry pi.
12. Add more images of your own in the asset folder and depth image will be genrated in depth output folder.
